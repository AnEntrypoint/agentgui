AgentGUI

Multi-agent GUI client for AI coding agents with real-time streaming, WebSocket sync, and SQLite persistence

View on GitHub

Features

⚡

Real-time Streaming

Watch agents work with live output, tool calls, and file operations in real-time.

🤖

Multi-Agent Support

Switch between Claude Code, Gemini CLI, OpenCode, Kilo, and more from one interface.

💾

Session Persistence

Full conversation history with SQLite database and session management.

🔄

WebSocket Sync

Live updates across multiple clients with automatic reconnection.

🎤

Voice Integration

Speech-to-text and text-to-speech support powered by Transformers.js.

🛠️

Tool Management

Install and update agent plugins directly from the UI.

Quick Start

npm install npm run dev # Server starts on http://localhost:3000/gm/

System Requirements

  • Node.js 18+
  • SQLite 3
  • Modern browser (Chrome, Firefox, Safari, Edge)

Architecture

server.js HTTP server + WebSocket + API routes database.js SQLite (WAL mode), schema, queries lib/claude-runner.js Agent framework - CLI process spawner lib/acp-manager.js ACP tool lifecycle management lib/speech.js STT/TTS via @huggingface/transformers static/js/client.js Main client logic static/js/streaming-renderer.js Event rendering
