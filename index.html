🤖 AgentGUI

Multi-agent GUI client for AI coding agents with real-time streaming

Get Started

✨ Features

🤖

Multi-Agent Support

Works with Claude Code, Gemini CLI, OpenCode, Goose, Kilo, Codex, and more. Auto-discovers installed tools.

📡

Real-Time Streaming

Live execution visualization with WebSocket sync. Watch your AI agents work in real-time.

💾

Persistent Storage

SQLite-based conversation and session history. Never lose your work or chat history.

🎤

Voice I/O

Built-in speech-to-text and text-to-speech powered by HuggingFace transformers.

📁

File Browser

Integrated file system explorer with drag-drop upload and syntax highlighting.

🔧

Tool Manager

Install and update agent plugins directly from the UI. One-click setup.

🚀 Quick Start

# Install globally
npm install -g agentgui

# Run the server
agentgui

# Or use npx
npx agentgui

Server starts on http://localhost:3000 and redirects to /gm/

🏗️ Architecture

Agent Framework

Spawns CLI processes, parses stream-json output, manages lifecycle

SQLite Storage

WAL mode, schema with conversations, messages, events, sessions

WebSocket Sync

Real-time bi-directional sync with subscribe/unsubscribe pattern

ACP Protocol

Auto-starts OpenCode/Kilo HTTP servers, health checks, crash recovery
